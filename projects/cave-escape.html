Cave Escape is a 2D platformer game for Windows that features 5 worlds filled with enemies, bosses, secrets, achievements, and over 60 different levels. It's a quick game meant to be played all the way through in one sitting, and can be played using your keyboard or an Xbox controller. This was my first video game, and I made it in GameMaker: Studio. The art was made using public domain clip art or drawing things in the program paint.net.
