The Escape Projects are a series of puzzle games designed to be solved as if the player is in an escape room. Each game features three locks with four-digit alphabet or number combinations that need to be correctly inputted to unlock. I also created an escape template site that will generate the code needed for a user to create their own escape site with the same layout as these eight games.
